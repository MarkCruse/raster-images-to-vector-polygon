Land Cover - Punta Gorda, FL

An interactive map with an overlay of land cover data

This map is the end result of processes created by Mark Cruse for wrangling together raster data images into vector polygons. The Python notebook and the Github repository provide insight into the processes.

Data Wrangling Notebook (HTML version): Convert NLCD Raster Images to Vector Polygons
Repository: MarkCruse/raster-images-to-vector-polygon
Data Source: Multi-Resolution Land Characteristics (MRLC) Consortium
Color Descriptions: National Land Cover Database Class Legend and Description
